# BSI IT-Grundschutz – Template-Satz (Markdown)

Dieser Template-Satz unterstützt die Erstellung der typischen Dokumentation für ein Informationssicherheitsmanagement nach **BSI IT-Grundschutz**.
Er orientiert sich an der gängigen Umsetzungspraxis entlang der BSI Standards (u. a. **200-1 Managementsystem**, **200-2 Methodik**, **200-3 Risikoanalyse**).

## Namenskonvention
`xxxx_ThemadesDokuments.md`  
- `xxxx` = 4-stellige Nummer zur Reihenfolge (in 10er-Schritten)

## Ordner
- `diagrams/` für Diagramme (Informationsverbund-Grenze, Netzplan, Datenflüsse, Modellierungsübersichten)
