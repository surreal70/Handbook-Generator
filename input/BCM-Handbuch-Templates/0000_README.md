# BCM-Handbuch – Template-Satz (Markdown)

Dieses Verzeichnis enthält **Template-Seiten** für ein **BCM-Handbuch / BCMS-Handbuch / Notfallhandbuch** in deutscher Sprache, orientiert an gängiger Praxis (z. B. ISO 22301/BSI-nahe Struktur, ohne Normtext zu kopieren).

## Namenskonvention
`xxxx_ThemadesDokuments.md`  
- `xxxx`: 4-stellige Nummer zur Reihenfolge (in 10er-Schritten)  
- `Thema`: sprechender Dateiname (Unterstriche statt Leerzeichen)

## Empfohlene Nutzung
1. Fülle zuerst Governance und Geltungsbereich (0010–0060).
2. Erstelle BIA & Risikoanalyse (0070–0100).
3. Leite Strategien ab und dokumentiere Pläne/Runbooks (0110–0200).
4. Plane Übungen/Tests und Pflegezyklen (0210–0250).
5. Nutze Anhänge für Checklisten, Kontakte, Vorlagen (0260+).

## Ordner
- `diagrams/` – Platzhalter für Diagramme (Abhängigkeiten, Entscheidungsbäume, Organigramme)

## Platzhalter
- `[TODO]` = auszufüllen
- `Option A/B/...` = auswählen / bereinigen
- Tabellen sind als Vorlage gedacht (kannst du in Markdown oder als Link zu Excel/Tool pflegen)
