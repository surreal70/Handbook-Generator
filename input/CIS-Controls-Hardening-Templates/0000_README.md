# CIS Controls – Hardening Template-Satz (Markdown)

Dieser Template-Satz unterstützt ein **Hardening-Programm** basierend auf **CIS Controls v8**.
Er dient dazu, **interne Hardening-Baselines** und **applikationsspezifische Standards** zu dokumentieren, nachzuweisen und zu betreiben.

## Struktur
- **Grundlagen** (0010–0050): Vorgehen, Rollen, Lifecycle, Ausnahmen, Tests
- **Betriebssysteme** (0100–0190): OS-Baselines je Plattform
- **Applikationen** (0200–0390): Standards/Baselines je Anwendungsklasse
- **Anhänge** (0400+): Mapping- und Evidence-Templates

## Namenskonvention
`xxxx_ThemadesDokuments.md` (xxxx = 4-stellige Reihenfolge)

## Hinweis zur Abgrenzung
- CIS Controls = organisatorische/technische **Kontrollen** (Was & Warum).
- Hardening-Baselines/Standards = konkrete **Konfiguration** (Wie).
Für konkrete Setting-Details kannst du zusätzlich CIS Benchmarks nutzen, aber **texte nicht kopieren**; verweise stattdessen auf interne Baselines oder externe Links.
