# ISMS-Template-Satz (ISO/IEC 27001) – Markdown

Dieses Verzeichnis enthält **Dokumententemplates** für ein **ISMS nach ISO/IEC 27001:2022** (inkl. Amendment 1:2024 als Referenzpunkt).  
Die **Policies** sind bewusst **abstrakt** formuliert; **Details** werden in **nachgelagerten Richtlinien/Standards/Prozessen** geregelt.

## Namenskonvention
`xxxx_ThemadesDokuments.md`  
- `xxxx`: 4-stellige Reihenfolge (10er-Schritte)
- Unterstriche statt Leerzeichen

## Aufbau-Idee
- 0010–0160: ISMS-Grundlagen (Kontext, Scope, Risk, Performance, Improvement)
- 0200–0700: Übliche InfoSec/IT-Policies (abstrakt) + abgeleitete Richtlinien (konkret)
- 0710+: Anhänge/Mapping/Inventare

## Ordner
- `diagrams/` – Diagramme (ISMS-Organigramm, Datenflüsse, Netzpläne, etc.)
