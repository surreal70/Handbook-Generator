# Framework Mapping Summary

## Overview

This document provides a comprehensive overview of all available framework templates in the handbook generator. The system currently supports 23 different frameworks across multiple domains including IT operations, security, compliance, governance, and quality management.

## Available Frameworks

### 1. IT Operations & Service Management
- **IT-Operation**: ITIL v4, ISO 20000-1:2018, COBIT 2019
- **Service Directory**: Service catalog and documentation templates

### 2. Information Security Management
- **ISMS**: ISO/IEC 27001:2022 (including Amendment 1:2024), Annex A Controls
- **BSI Grundschutz**: BSI Standards 200-1, 200-2, 200-3, IT-Grundschutz Compendium
- **CIS Controls**: CIS Controls v8.1 for system hardening and security baselines
- **NIST 800-53**: NIST SP 800-53 Rev. 5 Security and Privacy Controls
- **NIST CSF**: NIST Cybersecurity Framework 2.0

### 3. Business Continuity & Resilience
- **BCM**: ISO 22301:2019, BSI BCM Standards

### 4. Data Protection & Privacy
- **GDPR**: EU General Data Protection Regulation compliance
- **HIPAA**: Health Insurance Portability and Accountability Act

### 5. Financial & Audit Frameworks
- **SOC1**: Service Organization Control 1 (SSAE 18/ISAE 3402)
- **TSC**: Trust Services Criteria (SOC 2)
- **IDW PS 951**: German audit standard for IT systems
- **COSO**: Committee of Sponsoring Organizations Internal Control Framework

### 6. Industry-Specific Compliance
- **PCI DSS**: Payment Card Industry Data Security Standard
- **HIPAA**: Healthcare compliance
- **TISAX**: Trusted Information Security Assessment Exchange (Automotive)
- **DORA**: DevOps Research and Assessment metrics

### 7. Cloud Security
- **CSA CCM**: Cloud Security Alliance Cloud Controls Matrix

### 8. Quality & Governance
- **ISO 9001**: Quality Management Systems
- **ISO 31000**: Risk Management
- **ISO 38500**: IT Governance
- **TOGAF**: The Open Group Architecture Framework

### 9. Product Security
- **Common Criteria**: ISO/IEC 15408 Security Target documentation
