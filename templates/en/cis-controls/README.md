# CIS Controls v8 Templates

## Overview

This directory contains English templates for CIS Controls v8.

## Template Structure

Templates are numbered using a 4-digit prefix (NNNN) with 10-increment spacing:
- 0000: Metadata template
- 0010-0090: Foundation templates
- 0100+: Main content templates
- 9999: Framework mapping documentation

## Usage

These templates are used by the Handbook Generator to create CIS Controls v8 documentation.

For the German version, see: `templates/de/cis-controls/`
