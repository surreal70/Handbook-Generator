## Audit Objectives according to IDW PS 951

### Main Objectives

1. **Regularity of IT Systems**
   - Templates: 0300-0399 (IT Systems)
   - Audit of system documentation
   - Assessment of system controls

2. **Effectiveness of IT Controls**
   - Templates: 0200-0299 (IT Processes)
   - Control tests
   - Assessment of control effectiveness

3. **IT Governance and IT Organization**
   - Templates: 0100-0199 (IT Strategy)
   - Assessment of governance structure
   - Audit of responsibilities

4. **IT Security and Data Protection**
   - Templates: 0500-0599 (IT Security)
   - Security controls
   - Data protection compliance

5. **IT Operations and Availability**
   - Templates: 0400-0499 (IT Infrastructure)
   - Operations processes
   - Backup and recovery
