# Equipment Security

## Purpose

This document defines measures for securing equipment according to TISAX requirements.

## Scope

This document applies to all IT equipment and assets of {{ source.organization_name }}.

## Equipment Protection

### Location Security
- Secure placement of equipment
- Protection from physical access
- Theft protection
- Environmental protection

### Mobile Devices
- Hard disk encryption
- Strong authentication
- Remote wipe function
- Theft protection

### Maintenance
- Regular maintenance
- Documentation
- Authorized personnel
- Data deletion before repair

## Decommissioning

### Data Deletion
- Secure deletion of all data
- Multiple overwriting
- Certified destruction
- Documentation

### Disposal
- Environmentally friendly disposal
- Certified service providers
- Documentation
- Compliance with regulations

## TISAX-Specific Requirements

### VDA ISA Controls
- **5.4**: Protection against external and environmental threats
- **5.5**: Working in security areas
- **5.6**: Delivery and loading areas

### Assessment Evidence
- Equipment inventory
- Maintenance logs
- Disposal evidence

## Metrics

{{ source.organization_name }} measures:
- Number of stolen/lost devices
- Encryption compliance rate
- Number of proper disposals
