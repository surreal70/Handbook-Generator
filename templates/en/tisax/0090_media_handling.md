# Media Handling

## Purpose

This document describes the secure handling of storage media at {{ source.organization_name }}.

## Scope

This document applies to all physical and electronic storage media.

## Media Types

- Hard drives and SSDs
- USB drives and external drives
- CDs, DVDs, Blu-rays
- Magnetic tapes
- Paper documents
- Mobile devices

## Handling Guidelines

### Storage
- Secure storage according to classification
- Access control
- Environmental conditions (temperature, humidity)

### Transport
- Encryption of sensitive data
- Secure packaging
- Tracking

### Disposal
- Secure deletion of electronic media
- Physical destruction
- Certified disposal for confidential media

## Responsibilities

**Media Manager**: {{ source.media_manager }}
**Disposal Officer**: {{ source.disposal_officer }}
