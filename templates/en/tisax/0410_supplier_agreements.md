# Supplier Agreements

## Purpose

This document describes requirements for agreements with suppliers according to TISAX requirements.

## Scope

This document applies to all contracts with suppliers of {{ source.organization_name }}.

## Contract Components

### Security Requirements
**Mandatory Clauses:**
- Confidentiality agreement
- Data protection agreement
- Information security requirements
- Audit rights
- Incident reporting obligation
- Subcontractor regulations

### Service Level Agreements
- Availability: {{ source.sla_availability }}%
- Response Time: {{ source.sla_response_time }}
- Resolution Time: {{ source.sla_resolution_time }}
- Reporting requirements

### Liability and Insurance
- Liability limits
- Insurance evidence
- Compensation regulations
- Contractual penalties for violations

## Data Protection

### Data Processing
**When processing personal data:**
- Data Processing Agreement (DPA) according to GDPR
- Technical and organizational measures (TOMs)
- Authority to issue instructions
- Deletion obligations

## Compliance

### Certifications
**Required for critical suppliers:**
- ISO 27001
- TISAX (for automotive suppliers)
- Industry-specific certifications

### Audit Rights
- Right to audits
- Notice period
- Frequency
- Costs

## Contract Management

### Contract Term
- Duration
- Termination periods
- Extension options
- Exit clauses

### Contract Changes
- Written change requests
- Assessment of impact
- Approval
- Documentation

## Contract Termination

### Termination
**Reasons:**
- Contract expiration
- Termination
- Contract violation
- Insolvency

**Process:**
1. Notification
2. Handover/return of data
3. Revocation of access
4. Deletion of data
5. Final documentation

## TISAX-Specific Requirements

### VDA ISA Controls
- **8.2**: Supplier agreements

### Assessment Evidence
- Template contracts
- Current contracts
- DPAs
- Audit reports

## Metrics

{{ source.organization_name }} measures:
- Number of active supplier contracts
- Compliance rate
- Number of contract violations
