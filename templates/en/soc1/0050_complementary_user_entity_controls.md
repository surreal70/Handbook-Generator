## Purpose

This document describes the controls that must be present at user organizations to achieve the control objectives.

## Scope

This document covers:
- Definition of complementary controls
- Mapping to control objectives
- User organization responsibilities
- Impact on financial reporting

## Overview

### Concept of Complementary Controls

Complementary User Entity Controls (CUECs) are controls that must be implemented and operated by user organizations to achieve the service organization's control objectives.

### Responsibilities

**Service Organization**:
- Identification of required CUECs
- Communication of CUECs to user organizations
- Provision of implementation guidance

**User Organization**:
- Implementation of CUECs
- Operation and monitoring of CUECs
- Assessment of CUEC effectiveness
