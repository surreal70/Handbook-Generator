# Prinzip 6: Beste verfügbare Informationen

## Zweck

Dieses Dokument beschreibt das Prinzip "Beste verfügbare Informationen" des ISO 31000 Risikomanagements. Die Inputs zum Risikomanagement basieren auf historischen und aktuellen Informationen sowie auf zukünftigen Erwartungen.

## Prinzip-Beschreibung

Die Inputs zum Risikomanagement basieren auf historischen und aktuellen Informationen sowie auf zukünftigen Erwartungen. Risikomanagement berücksichtigt explizit alle Einschränkungen und Unsicherheiten im Zusammenhang mit solchen Informationen und Erwartungen.

### Kernaussagen

- Entscheidungen basieren auf besten verfügbaren Daten
- Historische Daten werden analysiert
- Aktuelle Informationen werden berücksichtigt
- Zukünftige Entwicklungen werden antizipiert
- Unsicherheiten werden explizit behandelt

## Umsetzung in {{ meta.organization }}

### Informationsquellen

| Informationstyp | Quellen |
|-----------------|---------|
| Historische Daten | Risikoregister, Vorfallsdatenbank, Audit-Berichte |
| Aktuelle Informationen | Monitoring-Systeme, KPIs, Berichte |
| Zukunftserwartungen | Marktanalysen, Trendstudien, Expertenmeinungen |
| Externe Daten | Branchenberichte, Regulatorische Updates |

### Datenqualität

**Qualitätskriterien:**
- **Genauigkeit**: Korrektheit der Informationen
- **Vollständigkeit**: Umfang der verfügbaren Daten
- **Aktualität**: Zeitnähe der Informationen
- **Relevanz**: Bedeutung für Entscheidungen
- **Zuverlässigkeit**: Vertrauenswürdigkeit der Quellen

### Umgang mit Unsicherheiten

**Methoden zur Unsicherheitsbehandlung:**
1. Sensitivitätsanalysen
2. Szenarioplanung
3. Monte-Carlo-Simulationen
4. Expertenschätzungen mit Konfidenzintervallen
5. Dokumentation von Annahmen und Einschränkungen

## Informationsmanagement

### Datensammlung und -analyse

- Systematische Erfassung relevanter Informationen
- Regelmäßige Aktualisierung der Datenbasis
- Analyse von Trends und Mustern
- Validierung von Informationen
- Dokumentation von Datenquellen

### Wissensmanagement

- Lessons Learned Datenbank
- Best Practice Sammlung
- Expertenwissen dokumentieren
- Erfahrungsaustausch fördern
- Kontinuierliche Wissensaktualisierung

## Entscheidungsfindung

### Informationsbasierte Risikoentscheidungen

Entscheidungen im Risikomanagement berücksichtigen:
- Verfügbare Daten und deren Qualität
- Bekannte Unsicherheiten und Einschränkungen
- Expertenmeinungen und Erfahrungen
- Kosten-Nutzen-Verhältnis von Informationsbeschaffung
- Zeitliche Dringlichkeit der Entscheidung
