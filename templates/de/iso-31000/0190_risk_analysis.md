# Risikoanalyse

## Zweck

Dieses Dokument beschreibt den Prozess der Risikoanalyse.

## Analysemethoden

### Qualitative Analyse

Bewertung mit Kategorien:
- Sehr niedrig / Niedrig / Mittel / Hoch / Sehr hoch

### Quantitative Analyse

Numerische Bewertung:
- Wahrscheinlichkeit in %
- Auswirkung in EUR
- Erwartungswert = Wahrscheinlichkeit × Auswirkung

### Semi-quantitative Analyse

Bewertungsskalen:
- Wahrscheinlichkeit: 1-5
- Auswirkung: 1-5
- Risikoscore: Wahrscheinlichkeit × Auswirkung

## Bewertungsdimensionen

| Dimension | Beschreibung |
|-----------|--------------|
| Eintrittswahrscheinlichkeit | Wie wahrscheinlich ist das Ereignis? |
| Auswirkung | Welche Konsequenzen hätte es? |
| Geschwindigkeit | Wie schnell tritt die Auswirkung ein? |
| Dauer | Wie lange dauert die Auswirkung an? |
