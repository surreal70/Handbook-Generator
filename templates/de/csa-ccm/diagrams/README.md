# CSA CCM Diagrams

This directory contains diagrams and visual representations for the CSA CCM handbook.

## Purpose

Diagrams help visualize complex concepts, processes, and relationships within the CSA CCM framework.

## Supported Formats

- **PlantUML** (.puml) - For process flows, sequence diagrams, component diagrams
- **Mermaid** (.mmd) - For flowcharts, Gantt charts, state diagrams
- **Draw.io** (.drawio) - For complex architecture diagrams
- **SVG/PNG** - Exported images for inclusion in handbooks

## Usage

Reference diagrams in handbook templates using:
```markdown
![Diagram Title](diagrams/diagram-name.png)
```

## Naming Convention

Use descriptive names with prefixes:
- `process-` for process flows
- `org-` for organizational charts
- `timeline-` for timelines and Gantt charts
- `flow-` for decision flows
- `arch-` for architecture diagrams
- `data-` for data flow diagrams
