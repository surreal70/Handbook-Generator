- Logs and protocols
- Memory images (RAM, hard drives)
- Network traffic
- Emails
- Documents
- Physical media

### Collection
**Methods:**
- Forensic copies
- Hash values for integrity assurance
- Write blockers for physical media
- Documentation of all steps
