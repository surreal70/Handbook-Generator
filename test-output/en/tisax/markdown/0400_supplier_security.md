## Purpose

This document defines requirements for security in supplier relationships according to TISAX requirements.

## Scope

This document applies to all suppliers and service providers of [TODO].

## Supplier Management

### Supplier Assessment
**Before Contract:**
- Security assessment
- Compliance review
- References
- Financial stability

**Criteria:**
- Information security management
- Data protection compliance
- Certifications (ISO 27001, TISAX)
- Incident response capabilities

### Supplier Categorization
**Critical Suppliers:**
- Access to confidential data
- Business-critical services
- Extended security requirements
- Regular audits

**Standard Suppliers:**
- Basic security requirements
- Sampling reviews

## Contractual Requirements

### Security Clauses
**Mandatory:**
- Confidentiality agreement (NDA)
- Data protection agreement
- Security requirements
- Audit rights
- Incident reporting obligation
- Liability regulations

- Availability
- Performance
- Security requirements

- Reporting
