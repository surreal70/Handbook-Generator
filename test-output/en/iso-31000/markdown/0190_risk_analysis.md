## Purpose

This document describes the risk analysis process.

## Analysis Methods

### Qualitative Analysis

Assessment with categories:
- Very low / Low / Medium / High / Very high

### Quantitative Analysis

Numerical assessment:
- Likelihood in %
- Impact in currency
- Expected value = Likelihood × Impact

### Semi-quantitative Analysis

Rating scales:
- Likelihood: 1-5
- Impact: 1-5
- Risk score: Likelihood × Impact

## Assessment Dimensions

| Dimension | Description |
|-----------|-------------|
| Likelihood | How probable is the event? |
| Impact | What consequences would it have? |
| Velocity | How quickly does the impact occur? |
| Duration | How long does the impact last? |
