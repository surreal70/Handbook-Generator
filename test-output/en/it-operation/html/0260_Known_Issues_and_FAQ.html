Files and Storage

Q: I accidentally deleted a file. Can it be recovered?

A: File recovery:

  1. Check recycle bin:
  2. Windows: Recycle Bin on desktop
  3. macOS: Trash in dock

  4. Linux: Trash folder

  5. Network drive:

  6. Check previous versions (Right-click → Properties → Previous Versions)

  7. Shadow copies available?

  8. Backup recovery:

  9. Create ticket (P3 - Medium)
  10. Provide filename, path, and approximate deletion date
  11. IT team restores from backup

Important: The sooner reported, the higher the success rate!
