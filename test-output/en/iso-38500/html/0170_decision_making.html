Purpose

This document describes the decision-making processes in IT governance.

Scope

This document applies to: - AdminSend GmbH - All IT governance decisions

Decision Types

Strategic Decisions

Examples: - IT strategy - Major investments - Risk tolerance

Decision Maker: Board

Process: 1. Analysis and assessment 2. Recommendation by Executive Management 3. Board decision 4. Communication

Tactical Decisions

Examples: - Project approvals - Resource allocation - Policies

Decision Maker: Executive Management, IT Governance Committee

Process: 1. Business case 2. Assessment 3. Committee decision 4. Implementation

Operational Decisions

Examples: - Technology selection - Process changes - Incident handling

Decision Maker: CIO, IT Management

Process: 1. Problem identification 2. Option assessment 3. Decision 4. Implementation

Decision Criteria

  • Strategic alignment
  • Cost-benefit ratio
  • Risks
  • Resource availability
  • Compliance
