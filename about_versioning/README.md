## Version Numbering

The project follows semantic versioning:
- **MAJOR.MINOR.PATCH** (e.g., 0.0.9)
- MAJOR: Breaking changes
- MINOR: New features (backwards compatible)
- PATCH: Bug fixes and minor improvements
